Get, add, and remove information regarding the course. 

Commands: 
add - adds info to the dictionary 
remove - removes info from the dictionary
lookup - search for specific key
all - prints all information avaliable
list - lists all keys you can look up
tenta [course ID] - search examination on the specified course id (ex D0020E)

usage in chat(skip /[yourCommand] when talking directly to the bot);
/[yourcommand] info |remove/printAll/list] or
/[yourcommand] info add [name] [value] (for add only)

example:
/zoombot info add kurskod D0020E
